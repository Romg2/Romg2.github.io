# 연습 문제 4.1.1

(1) 임의로 두 개의 시리즈 객체를 만든다. 모두 문자열 인덱스를 가져야 하며 두 시리즈에 공통적으로 포함되지 않는 라벨이 있어야 한다.


(2) 위에서 만든 두 시리즈 객체를 이용하여 사칙 연산을 한다.


```python

s2 = pd.Series([4, 5, 6, 7, 8],
            index = ["A", "B", "C", "D", "F"])

# (2) 시리즈 사칙연산
s3 = s1 * s2
s3[s3.notnull()]
```

**[문법]**

`DISTINCT`는 중복된 데이터를 제외하고 출력 한다.

```sql
SELECT DISTINCT <열 이름1>
FROM <테이블 이름>;
```

**[예시]**

```sql
SELECT DISTINCT JOB
FROM EMP;
```

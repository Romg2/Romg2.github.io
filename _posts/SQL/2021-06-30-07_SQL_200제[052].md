---

**[예시]**

`ROLLUP`은 마지막 행에 집계 결과를 출력한다.

다음 예시와 같이 `GROUP BY`절에 사용한다.

```sql
SELECT JOB, SUM(SAL) AS 합계
FROM EMP
GROUP BY ROLLUP(JOB);
```


**[결과]**

JOB|합계
|-|-|
ANALYST|6000
CLERK|4150
MANAGER|8275
PRESIDENT|5000
SALESMAN|5600
||29025


**[예시2]**

다음과 같이 여러 열을 지정하여 사용 가능하다.
```sql
SELECT DEPTNO, JOB, SUM(SAL) AS 합계
FROM EMP
GROUP BY ROLLUP(DEPTNO, JOB);
```

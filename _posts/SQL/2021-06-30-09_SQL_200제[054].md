**[예시]**

`GROUPING SETS`은 집계하고 싶은 열을 직접 지정한다.

다음 예시는 DEPTNO(부서번호)별 집계, JOB(직업)별 집계, 전체 집계를 출력한다.

```sql
SELECT DEPTNO, JOB, SUM(SAL) AS 합계
FROM EMP
GROUP BY GROUPING SETS( (DEPTNO), (JOB), () )
ORDER BY DEPTNO, JOB;
```

FROM EMP

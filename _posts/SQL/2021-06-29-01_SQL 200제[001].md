**[문법]**

테이블에서 특정한 열을 선택할 때는 다음과 같이 작성한다.


```sql
SELECT <열 이름1>, <열 이름2>, ...
FROM <테이블 이름>;
```

**[예시]**

```sql
SELECT EMPNO, ENAME, SAL
FROM EMP;
```

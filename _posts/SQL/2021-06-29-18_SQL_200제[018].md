`LENGTH`를 이용해서 문자열의 길이를 출력 가능하다.

**[예시]**

```sql
SELECT ENAME, 
       LENGTH(ENAME) AS 이름길이,
       LENGTH('한글의길이는?') AS 주석
FROM EMP;
```


**[결과]**

ENAME|이름길이|주석
|-|-|-|
KING|4|7
BLAKE|5|7
CLARK|5|7
JONES|5|7
MARTIN|6|7
ALLEN|5|7
TURNER|6|7
JAMES|5|7
WARD|4|7
FORD|4|7
